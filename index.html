What i do

My Services

Адаптивная вёрстка и UI

Верстаю адаптивные лендинги на HTML и CSS, использую Flex и Grid, чтобы интерфейс выглядел аккуратно и удобно на разных экранах.

Responsive Layouts & UI

I build responsive landing pages with HTML and CSS, using Flex and Grid to ensure interfaces look clean and convenient across different screens.

Адаптивный дизайн

Настраиваю адаптив под основные ширины: десктоп, планшет и мобильный. Подбираю медиазапросы и сетку так, чтобы блоки не ломались и сохраняли логику макета.

Responsive Design

I configure responsive design for main screen sizes: desktop, tablet, and mobile. I select media queries and grid layouts so blocks don't break and maintain the design logic.

Детали UI и стили

Работую с типографикой, цветами, иконками и небольшими эффектами (тени, hover, фокус), чтобы интерфейс выглядел живее и не был “холодным”.

UI Details & Styles

I work with typography, colors, icons, and small effects (shadows, hover, focus) to make interfaces feel alive and not "cold".

Основы JavaScript

Осваиваю базу JavaScript: условия, функции и логические задачи в браузере. Внедряю интерактивные элементы в свои проекты, постепенно переходя к более сложным задачам и динамическим сценариям.

JavaScript Basics

Mastering JavaScript fundamentals: conditions, functions, and browser-based logic. I am adding interactivity to my projects, gradually taking on more complex tasks and dynamic features.

Git и рабочий процесс

Использую Git для ведения истории изменений: делаю коммиты, работаю в ветке develop и собираю проекты в репозиториях, чтобы код развивался аккуратно и без хаоса.​

Git & Project Workflow

I use Git for change history: make commits, work in develop branch, and organize projects in repositories so code develops cleanly without chaos.
